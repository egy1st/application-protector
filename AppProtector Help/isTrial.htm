NotLicensed

NotLicensed is public variable indicates the state of your product license, so you can do the proper action
it will be True if  the protected product is not registered  or within trial period .

Syntax:

Public NotLicensed As Boolean = False

 

Example:

Dim MyProtection As New DynamicComponents.AppProtector()

If  MyProtection.NotLicensed Then Exit Sub

 

Note:

There is no need to end your application programmaticaly with the aid of  detecting the value NotLicensed, as AppProtector exits application autmatically when trial perid ends.
