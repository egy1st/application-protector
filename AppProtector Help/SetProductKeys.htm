SetProductKeys Function

While SetAlgorithms function defines the method DC.AppProtecor adopts to generate its activation keys, SetProductKeys function makes each activation key of these keys valid for any number of product of yours by adding a fingerprint for each combination of cmpany, product and its version.


Syntax:

SetProductKeys(ByVal Company_ID As String, ByVal Product_ID As String, ByVal Product_Version As String)

Example:

MyProtection.SetProductKeys("EgyFirst", "AppProtector", "V 3.5")
