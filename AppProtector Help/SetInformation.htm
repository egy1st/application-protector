SetInformation Function

SetInformation function enables you to  Define your customized message appears on the protection dialog box.
it include 2 parameters

  1. Product & Company info
  2. URL where customers can purchase your product by clicking Buy Now Button on the protection dialog box

Syntax:

SetInformation(ByVal BuyNow_URL As String, ByVal Message As String)

Example:

Dim CompanyInfo As String
Dim buynow_URL As String

CompanyInfo = "Application Protector V. 3.5" + vbCrLf                 'vbcrlf forces new line
CompanyInfo += "Powered by EgyFirst Software, Inc." + vbCrLf
CompanyInfo += "Free 30 Days Trial Version"

buynow_URL = "http://www.mygoldensoft.com/ordernow.htm"

MyProtection.SetInformation(buynow_URL, CompanyInfo)
